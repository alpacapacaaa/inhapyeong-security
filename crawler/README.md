# 인하대학교 강의 데이터 크롤링 계획서

이 폴더(`crawler/`)는 인하대학교 수강신청 시스템에서 최신 강의 정보를 수집하고 정제하기 위한 전용 모듈입니다.

## 1. 아키텍처 구성
- **src/init_depts.py**: 학과 코드 및 카테고리(교양/전공) 초기화 데이터 수집
- **src/course_scraper.py**: Selenium을 이용한 학과별 강의 목록 수집 (학수번호, 강의명, 교수 등)
- **src/syllabus_scraper.py**: 강의계획서 상세 내용(평가 비중, 교재 등) 수집 및 파싱
- **data/**: 크롤링된 중간 결과물 (JSON/CSV) 저장소

## 2. 수집 전략
1.  **학과 코드 추출**: `ddlDept` 셀렉트 박스에서 현재 운영 중인 모든 학과 코드를 저장합니다.
2.  **강의 목록 수합**: 학과 코드별로 검색 버튼을 클릭하고 결과 테이블을 파싱합니다.
3.  **라벨링 자동화**:
    - 교양: `ddlKita` 파라미터 값에 따라 핵심교양 영역 자동 매핑
    - 전공: 학과 코드와 학수번호 매칭을 통한 전필/전선 구분 로직 구현
4.  **강의계획서 파싱**: 팝업 URL에서 평가 비율(중간/기말/과제/출석)과 사용 교재 키워드를 정규표현식으로 추출합니다.

## 3. 실행 환경 및 가상환경(venv) 관리

### 3.1 초기 설정 (최초 1회) 
#### mac 환경에 맞춰져 있습니다! window사용자는 다른 방법을 찾아 사용하셔야 합니다.
```bash
cd crawler
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3.2 다시 시작하기 / 가상환경 켜기
크롤러를 다시 실행할 때는 항상 가상환경을 먼저 켜주어야 합니다.
```bash
cd crawler
source venv/bin/activate
```
*(터미널 앞에 `(venv)`가 표시되면 성공입니다)*

### 3.3 가상환경 나가기 (종료)
작업이 끝난 후 가상환경을 종료하려면 아래 명령어를 입력합니다.
```bash
deactivate
```

### 3.4 크롤러 실행
```bash
# 학과 초기화 (필요시)
python3 src/init_depts.py
# 강의 목록 수집
python3 src/course_scraper.py
# 강의계획서 상세 수집
python3 src/syllabus_scraper.py
```

## 4. 주의사항
- **서버 부하 방지**: `time.sleep(1)` 등을 활용해 요청 사이의 간격을 유지합니다.
- **예외 처리**: 강의계획서가 없거나 시스템 점검 시간일 경우에 대비한 `try-except` 로직을 강화합니다.
